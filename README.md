# CURED: Creating, Understanding, and Repairing Errors Demonstrator

This demo showcases recent research on errors in tabular data. Specifically, it shows how [tab-err](https://dl.acm.org/doi/10.1145/3774914) can be used to create errors in tables, how [MechDetect](https://arxiv.org/abs/2512.04138) can be used to understand error mechanisms in tables, and how [Conformal Data Cleaning](https://proceedings.mlr.press/v238/jager24a.html) can repair errors in tables.

### Usage

Ensure you have astral uv on your system for local running.

#### Running Locally

1. `uv sync`
2. `source .venv/bin/activate`
3. `python3 main.py`
